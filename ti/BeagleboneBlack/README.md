

Standard FTDI Cable 
===================

- Adafruit 4 Pin Cable (PL2303)

  | **Board**    |  **cable**  |
  |:------------:|:-----------:|
  | Pin 1 (GND)  | Black (GND) |
  | Pin 4 (RX)   | Green (TX)  |
  | Pin 5 (TX)   | White (RX)  |
